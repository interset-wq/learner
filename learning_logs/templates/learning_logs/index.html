Data Model

ER diagram of the learning logs data structure:

erDiagram
    Topic {
        int id PK
        varchar text
        datetime date_added
        int owner_id FK
        bool is_public
    }

    Entry {
        int id PK
        varchar title
        text text
        datetime date_added
        int topic_id FK
        bool is_public
    }

    Comment {
        int id PK
        int entry_id FK
        int user_id FK
        int parent_id FK
        text text
        datetime created_at
    }

Features

Markdown Editor

Write entries in Markdown with EasyMDE. Supports code blocks, tables, images, and live preview.

Social Interactions

Like entries, leave comments with @mentions, and share links. Powered by HTMX for real-time updates.

Comment Threads

Nested replies (max 2 levels), click-to-reply with auto @mention, collapsible threads, infinite scroll.

Explore

Discover public topics and entries from other users. Browse the community learning feed.

Visibility Control

Choose public or private for each topic and entry. Private content is only visible to the owner.

HTMX Powered

Likes, comments, and infinite scroll work without page reloads. Minimal JavaScript, maximum interactivity.
