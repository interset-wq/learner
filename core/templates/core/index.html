Features

Class-Based Views

Uses Django's ListView and DetailView for clean, reusable view logic.

Atomic Voting

Uses F() expressions to prevent race conditions during vote counting.

Result Visualization

Displays vote counts with progress bars for each choice.

Pagination

Question list supports pagination for large datasets.
